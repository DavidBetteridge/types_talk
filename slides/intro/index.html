The trick is know that the digits 1-9 add up to 45. So each row,column and box must add to 45. If the centre box was say 1, then the 4 pairs must add to 44, ie 11 each. Terminology

Originally I thought I would explain all the terms first, and then look at the languages one by one. However, these here are just some of the categories and not all languages agree on the definitions.

What are types?

A fair question is however, what are types? If I asked around the room I would get a number of different answers. For the purposes of this talk.

Primitive Types

We have primitive types, such as numbers (various forms), booleans, strings and maybe dates

Complex types
From the primitive types we can construct more complex types

Type Alias
Which we can then name. Called a type alias. For c-sharp, java programmers - these are more like interfaces than classes as they don't have Implementation. In swift however.... both these primitive types and complex types are called Named Types. A type without a name (such as a tuple) is called a compound type. We can already see there is a lot of variation in the terminology.
